# Hi there! Welcome to the SSR Little server!

## You want to execute me?

### Using a CSR Approach

```bash
yarn start
```

### Using a SSR Approach

#### Create the app build for the client

```bash
yarn run build
```

#### Build the server code

```bash
yarn run ssr:build-server
```

#### Run the server!!

```bash
yarn run ssr:start
```
